# 校园网自动连接系统使用说明

## 说明

   确保自己离开宿舍的时候电脑可以保持联网，联网逻辑用playwright写的，连接的时候有点卡，可以试一下自己重写逻辑

## 安装配置

### 1. 安装依赖
```bash
pip install -r requirements.txt
playwright install
```

### 2. 配置参数
编辑`setting.yaml`进行配置：

```yaml
# 配置信息 - 请修改为你的实际信息
校园网账号: "账号"           # 你的校园网账号
校园网密码: "密码"           # 你的校园网密码
url: "URL"                  # 这个是认证校园网时的完整路径，需要全部复制粘贴
检测间隔: 600
```
